# maskrcnn_telcom_custom_object_detection
Detection on various types of objects on top of the tower (e.g.: GPS, Cable, Sign Board, Jumper Labeling and etc)


## Problem Statement 1
GPS units critical for network synchronization. Weatherproofing cables must be properly installed to prevent signal degradation and equipment failure during monsoons.

## Our Solution
AI model detects GPS antenna presence and verifies weather-proof cable connections in tower site images. Identifies missing or improperly installed protection.

## Business Impact
• Prevents network timing issues
• Reduces equipment failure significantly
• Eliminates manual site inspections
• Proactive maintenance alerts
